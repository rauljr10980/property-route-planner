# How to Find Your Google Cloud Project

## Method 1: Project Dropdown (Top of Page)

1. **Open Google Cloud Console**
   - Go to: https://console.cloud.google.com/

2. **Look at the top of the page**
   - You'll see a **project dropdown** at the very top (next to "Google Cloud")
   - It will either show:
     - Your current project name (e.g., "property-tax-tracker")
     - Or "Select a project" if no project is selected

3. **Click the dropdown**
   - You'll see a list of all your projects
   - Click on the project you want to use
   - Or click "New Project" if you need to create one

---

## Method 2: Project Selector Page

1. **Direct link to project selector:**
   - https://console.cloud.google.com/home/dashboard?project=

2. **Or click:**
   - The project dropdown at the top
   - Click "All" at the bottom
   - This shows all your projects

---

## Method 3: Resource Manager

1. **Go to Resource Manager:**
   - https://console.cloud.google.com/cloud-resource-manager

2. **You'll see:**
   - All your projects listed
   - Project name, ID, and number
   - Filter/search options

---

## Method 4: Check Current Project

To see what project you're currently in:

1. Look at the **top-left corner** of any page
2. It will show your current project name
3. Click it to switch projects

---

## Can't Find Your Project?

### If you haven't created one yet:
1. Click the project dropdown
2. Click **"New Project"**
3. Name it: `property-tax-tracker`
4. Click **"Create"**

### If you see "No projects":
1. Make sure you're signed in with the right Google account
2. Check if billing is enabled
3. You might need to create a new project

### If you have multiple Google accounts:
1. Check which account you're signed in with (top-right)
2. Switch accounts if needed
3. Projects are account-specific

---
